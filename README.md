# Site – Freire Júnior Caiado – Advocacia e Consultoria

Site estático publicado via GitHub Pages, com versões em português (PT) e inglês (EN).

## Estrutura

- `index.html` – versão principal em português (pt-BR)
- `index-en.html` – versão em inglês
- `favicon.svg` – ícone do site
- `og-freirejcaiado.png` – imagem para pré-visualização (Open Graph)
- `robots.txt` – instruções para robôs de busca
- `sitemap.xml` – sitemap com as URLs principais
- `404.html` – página de erro 404 amigável

## Publicação

O site é publicado em:

- `https://freirejrcaiado.github.io/freire-junior-caiado-site/` (PT)
- `https://freirejrcaiado.github.io/freire-junior-caiado-site/index-en.html` (EN)

Para atualizar o site:

1. Fazer ajustes nos arquivos `.html`, imagens ou demais recursos.
2. Fazer commit no repositório.
3. Fazer push para a branch configurada no GitHub Pages (geralmente `main`).

As alterações são refletidas automaticamente após alguns instantes.

## Observações éticas

Todo o conteúdo do site é estritamente informativo e está em conformidade
com o Estatuto da Advocacia e da Ordem dos Advogados do Brasil, bem como com
o Código de Ética e Disciplina da OAB, especialmente no âmbito das Seccionais
do Distrito Federal e da Bahia, sem promessa de resultado ou captação indevida
de clientela.
